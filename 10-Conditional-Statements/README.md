# 10. Conditional Statements

Conditional statements allow a program to make decisions based on specific conditions. Instead of executing every statement in sequence, a program can choose different execution paths depending on whether a condition is `True` or `False`.

Decision making is one of the most important concepts in programming. From checking whether a user is old enough to vote to validating a password or determining a student's grade, conditional statements make programs intelligent and interactive.

This module introduces Python's conditional statements and demonstrates how they are used to control program flow using simple, readable, and practical examples.

# Why Conditional Statements Matter

Almost every software application makes decisions.

Examples include:

- Checking whether a password is correct
- Determining whether a user can log in
- Calculating grades
- Verifying voting eligibility
- Detecting positive or negative numbers
- Comparing values
- Classifying data
- Making recommendations

Without conditional statements, programs would execute every statement in the same order without adapting to different situations.

# Best Practices

While learning conditional statements, remember the following guidelines:

- Write clear and readable conditions.
- Use meaningful variable names.
- Indent code blocks consistently.
- Keep conditional logic simple whenever possible.
- Test programs using different input values.
- Use logical operators only when necessary.
- Avoid unnecessary nesting.
- Add comments when they improve understanding.
