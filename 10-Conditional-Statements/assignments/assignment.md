# Assignment Overview

This assignment is designed to help you practice Python conditional statements by solving real-world programming problems.

You should complete each question using only the concepts learned up to this module.

## Concepts Covered

- `if` statement
- `if-else` statement
- `if-elif-else` statement
- Nested `if` statement
- Comparison operators
- Logical operators
- User input
- Variables
- Type conversion

---

# Instructions

- Write each program in a separate Python file.
- Use meaningful variable names.
- Follow Python's indentation rules.
- Test your programs with different inputs.
- Do not use loops, functions, lists, dictionaries, classes, or exception handling.

# Beginner Level

## Question 1 — Positive, Negative, or Zero

Write a program that accepts an integer from the user and displays whether it is:

- Positive
- Negative
- Zero

---

## Question 2 — Even or Odd

Write a program that accepts an integer and determines whether it is even or odd.

---

## Question 3 — Voting Eligibility

Write a program that asks the user's age and displays:

- "Eligible to vote" if the age is 18 or above.
- "Not eligible to vote" otherwise.

---

## Question 4 — Pass or Fail

Write a program that asks the user to enter marks.

Display:

- "Pass" if marks are 40 or greater.
- "Fail" otherwise.

Also validate that the marks are between **0 and 100**.

---

## Question 5 — Age Category

Write a program that classifies a person's age into one of the following categories:

- Child (0–12)
- Teenager (13–19)
- Adult (20–59)
- Senior Citizen (60 and above)

Display an error message for negative ages.

# Intermediate Level

## Question 6 — Largest of Three Numbers

Write a program that accepts three numbers and displays the largest number.

## Question 7 — Grade Calculator

Write a program that accepts marks and displays the corresponding grade.

| Marks | Grade |
|-------:|:-----:|
| 90–100 | A |
| 80–89 | B |
| 70–79 | C |
| 60–69 | D |
| Below 60 | F |

Display an error message if marks are outside the range **0–100**.

## Question 8 — Leap Year Checker

Write a program that asks the user to enter a year.

Use conditional statements to determine whether the year is a leap year.

**Hint:** A leap year is:
- Divisible by 400, **or**
- Divisible by 4 but **not** divisible by 100.
