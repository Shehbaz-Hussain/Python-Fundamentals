# Assignment: Student Information Management System

## Problem Statement

Design a simple Python program that manages student information using variables. The program should store a student’s basic details, display them clearly, and allow the user to update a value such as the student’s age or grade.

---

## Objectives

By completing this assignment, students will:

- practice creating and using variables
- learn how to reassign values
- use descriptive variable names
- format output clearly
- understand how variables help manage data

---

## Functional Requirements

Your program should:

1. Store the following student information using variables:
   - `student_name`
   - `student_age`
   - `student_grade`
   - `student_city`

2. Print the student information in an organized format.

3. Allow the user to update at least one variable, such as `student_age`.

4. Print the updated information after the change.

5. Use clear and meaningful variable names.

---

## Constraints

- Use Python variables only for storing data.
- Follow PEP 8 naming style.
- Write clean and readable code.
- The program should be beginner-friendly and easy to understand.

---

## Sample Input

```python
student_name = "Aisha"
student_age = 20
student_grade = "A"
student_city = "Lahore"
```

---

## Sample Output

```python
Student Information
Name: Aisha
Age: 20
Grade: A
City: Lahore
```

After updating the age:

```python
Student Information
Name: Aisha
Age: 21
Grade: A
City: Lahore
```

---

## Bonus Tasks

- Add a variable for `student_id`.
- Display the information using a formatted string.
- Add a second update such as changing the student’s city.
- Use constants for values that should not change.
