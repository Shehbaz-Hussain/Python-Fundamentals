# Python Variables Exercises

This file contains a structured practice set for learning Python variables.

---

## Beginner Exercises (30)

1. Create a variable named `name` and assign your name to it.
2. Print the value of a variable named `city`.
3. Create a variable `age` and store your age.
4. Assign the value `100` to a variable named `score`.
5. Print a variable containing a string value.
6. Create two variables, `first_name` and `last_name`, and print both.
7. Store the value `True` in a variable named `is_student`.
8. Create a variable called `price` and assign `19.99` to it.
9. Assign `10` to `a` and `20` to `b`, then print both.
10. Create a variable called `message` and print it.
11. Assign the number `7` to a variable and print it.
12. Create a variable `favorite_color` and assign a color name.
13. Store your height in a variable and print it.
14. Create a variable `product` and assign the value `"book"`.
15. Assign `0` to a variable called `counter`.
16. Create a variable `temperature` and print it.
17. Store a float value in a variable and print it.
18. Create a variable `result` and assign the sum of two numbers.
19. Create a variable `greeting` and print a friendly message.
20. Create a variable `year` and assign `2026`.
21. Assign the same value to two variables.
22. Create a variable `status` and store `"active"`.
23. Print three variables in one line.
24. Create a variable `discount` and assign `15`.
25. Store the string `"Python"` in a variable.
26. Create a variable `number_of_days` and assign `7`.
27. Assign a boolean value to `is_active`.
28. Create a variable `weight` and print it.
29. Store your country name in a variable.
30. Create a variable `pi` and assign `3.14`.

## Intermediate Exercises (20)

1. Write a program that stores the price and quantity of an item and calculates total cost.
2. Create three variables and assign values using multiple assignment.
3. Swap the values of two variables and print the result.
4. Use variables in an arithmetic expression to compute the area of a rectangle.
5. Create variables for `name`, `age`, and `city`, then print them in one sentence.
6. Store a user’s first name and last name in separate variables.
7. Use `type()` to print the type of a variable.
8. Create a variable and reassign it to a different value.
9. Use variables to calculate simple interest.
10. Store a temperature in Celsius and convert it to Fahrenheit using variables.
11. Create variables for `base` and `height`, then calculate the area of a triangle.
12. Use multiple assignment to assign values to `x`, `y`, and `z`.
13. Create a variable `full_name` by combining two smaller variables.
14. Use variables to compute the average of three values.
15. Store a list of three numbers in a variable and print the second element.
16. Create a variable `discount_rate` and calculate a discounted price.
17. Create a variable `remaining_days` and reduce it by one.
18. Use variables to calculate the perimeter of a rectangle.
19. Create a variable `is_valid` and assign a boolean based on a condition.
20. Use variables to print a formatted message with several values.

## Advanced Exercises (10)

1. Build a small program that stores student information and prints a report.
2. Create a program that calculates the final grade from multiple subject scores.
3. Use variables to simulate a bank account balance and update it after a deposit.
4. Create a program that stores product information and calculates total revenue.
5. Build a mini calculator using variables for `num1`, `num2`, and `operator`.
6. Use variables to model a shopping cart with item prices and quantities.
7. Create a program that stores and updates a countdown timer using variables.
8. Build a simple inventory system using variables to track stock levels.
9. Use variables to calculate compound interest over multiple years.
10. Create a program that stores and analyzes monthly expenses.

---

## Tips for Practice

- Start with the beginner set and move upward gradually.
- Try to solve each exercise without looking at the solutions.
- Write your own examples after finishing each section.
- Focus on readability and naming clarity.
