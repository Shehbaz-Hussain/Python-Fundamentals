# Python Variables Image Plan

This document outlines an educational image plan for the Python Variables module. The images are intended to support beginner understanding and improve visual learning.

---

## 1. Variable Memory Diagram

- **Filename:** `variable-memory-diagram.png`
- **Purpose:** Show how a variable stores a value in memory.
- **Placement:** Near the section on What Is a Variable?
- **Detailed AI Image Prompt:** Create a clean educational diagram showing a labeled box named `name` containing the value `Alice`, with a small memory icon and a simple arrow indicating storage.
- **Recommended Dimensions:** 1600 x 900 px
- **Alt Text:** Diagram showing a variable named name storing the value Alice.
- **Caption:** A variable acts like a labeled box that holds a value.
- **SEO Filename:** `python-variable-memory-diagram.png`
