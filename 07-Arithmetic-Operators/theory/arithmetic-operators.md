# Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations in Python.

| Operator | Name | Example |
|---|---|---|
| + | Addition | 5 + 3 |
| - | Subtraction | 5 - 3 |
| * | Multiplication | 5 * 3 |
| / | Division | 5 / 3 |
| // | Floor Division | 5 // 3 |
| % | Modulus | 5 % 3 |
| ** | Exponentiation | 5 ** 3 |

## Example

```python
number1 = 10
number2 = 5

print(number1 + number2)
print(number1 - number2)
print(number1 * number2)
print(number1 / number2)
print(number1 // number2)
print(number1 % number2)
print(number1 ** 2)
