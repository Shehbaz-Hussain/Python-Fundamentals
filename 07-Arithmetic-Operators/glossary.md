# Glossary

## Module

**07 – Arithmetic Operators**

---

# Introduction

This glossary defines the key terms introduced in the **Arithmetic Operators** module. Understanding these terms will help you read, write, and reason about Python expressions more effectively.

---

# Addition

The mathematical operation of combining two or more values into a single total.

**Example**

```python
10 + 5
```

**Result**

```text
15
```

---

# Arithmetic Expression

A combination of operands and arithmetic operators that Python evaluates to produce a result.

**Example**

```python
(10 + 5) * 2
```

---

# Arithmetic Operator

A symbol used to perform mathematical calculations on numeric values.

Python arithmetic operators include:

- `+`
- `-`
- `*`
- `/`
- `//`
- `%`
- `**`

---

# Assignment Operator

The `=` operator used to assign a value to a variable.

**Example**

```python
total = 100
```

---

# Division

An arithmetic operation that divides one number by another using the `/` operator.

In Python, division always returns a floating-point number.

**Example**

```python
9 / 2
```

**Result**

```text
4.5
```

---

# Exponentiation

The operation of raising a number to the power of another number using the `**` operator.

**Example**

```python
2 ** 4
```

**Result**

```text
16
```

---

# Expression

A valid combination of values, variables, operators, and function calls that Python evaluates to produce a value.

**Example**

```python
price * quantity
```

---

# Floor Division

Division that returns the largest integer less than or equal to the exact quotient by discarding the fractional part.

It uses the `//` operator.

**Example**

```python
9 // 2
```

**Result**

```text
4
```

---

# Float

A numeric data type used to represent decimal (floating-point) numbers.

**Examples**

```python
3.14
5.0
-12.75
```

---

# Integer

A whole number without a decimal point.

**Examples**

```python
5
0
-12
```

---

# Modulus

The operation that returns the remainder after division.

It uses the `%` operator.

**Example**

```python
10 % 3
```

**Result**

```text
1
```

---

# Multiplication

The arithmetic operation of repeatedly adding a number to itself.

It uses the `*` operator.

**Example**

```python
6 * 7
```

**Result**

```text
42
```

---

# Operand

A value or variable on which an operator performs an operation.

**Example**

```python
10 + 5
```

Here:

- `10` is an operand.
- `5` is an operand.
- `+` is the operator.

---

# Operator

A symbol that tells Python to perform a specific operation.

**Examples**

```python
+
-
*
/
%
//
**
=
```

---

# Operator Precedence

The rules that determine the order in which operators are evaluated in an expression.

**Example**

```python
10 + 5 * 2
```

Python evaluates:

```text
5 × 2 = 10

10 + 10 = 20
```

---

# Parentheses

Grouping symbols `()` used to control the order of evaluation in expressions.

**Example**

```python
(10 + 5) * 2
```

---

# Quotient

The result obtained after dividing one number by another.

**Example**

```python
20 / 4
```

**Quotient**

```text
5.0
```

---

# Remainder

The value left after performing division.

**Example**

```python
10 % 4
```

**Result**

```text
2
```

---

# Subtraction

The arithmetic operation of finding the difference between two values.

It uses the `-` operator.

**Example**

```python
15 - 8
```

**Result**

```text
7
```

---

# Variable

A named storage location that holds data.

Variables allow programs to store and manipulate values.

**Example**

```python
age = 20
```

Here:

- `age` is the variable.
- `20` is the stored value.

---

# Numeric Data Type

A data type used to represent numbers.

The most common numeric data types in Python are:

- `int`
- `float`
- `complex`

For this module, the focus is on:

- `int`
- `float`

---

# Power

The result of raising a number to an exponent.

**Example**

```python
3 ** 3
```

**Result**

```text
27
```

---

# Mathematical Formula

An equation used to calculate a value using arithmetic operations.

**Example**

Rectangle area:

```python
area = length * width
```

---

# User Input

Data entered by a user while a program is running.

**Example**

```python
age = int(input("Enter your age: "))
```

---

# Type Conversion

The process of converting one data type into another.

Common numeric conversions:

```python
int()

float()
```

**Example**

```python
number = int(input("Enter a number:"))
```
