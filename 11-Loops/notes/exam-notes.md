# Exam Notes: Loops in Python

## Module Overview

Loops allow a program to execute the same block of code multiple times. They help automate repetitive tasks, reduce duplicate code, and make programs more efficient and easier to maintain.

Python supports two loop structures:

- `for` loop
- `while` loop

# for Loop

## Definition

A `for` loop is used when the number of iterations is known or when iterating over a sequence generated by `range()`.

## Syntax

```python
for variable in range(start, stop, step):
    # Code block
```

## Example

```python
for number in range(1, 6):
    print(number)
```

---

# while Loop

## Definition

A `while` loop repeatedly executes a block of code as long as a condition evaluates to `True`.

## Syntax

```python
while condition:
    # Code block
```

## Example

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

# The `range()` Function

The `range()` function generates a sequence of integers.

| Syntax | Description |
|---------|-------------|
| `range(stop)` | Starts from `0` and ends before `stop` |
| `range(start, stop)` | Starts from `start` and ends before `stop` |
| `range(start, stop, step)` | Uses a custom increment or decrement |

### Examples

```python
range(5)
```

Produces:

```
0 1 2 3 4
```

```python
range(1, 6)
```

Produces:

```
1 2 3 4 5
```

```python
range(10, 0, -2)
```

Produces:

```
10 8 6 4 2
```

> **Important:** The stop value is **not included**.

# Nested Loops

A nested loop is a loop inside another loop.

### Example

```python
for row in range(3):
    for column in range(3):
        print("*", end="")
    print()
```

### Common Uses

- Pattern printing
- Multiplication tables
- Row-and-column processing

---

# Loop Control Statements

## `break`

Terminates the nearest loop immediately.

### Example

```python
for number in range(1, 11):
    if number == 6:
        break

    print(number)
```

---

## `continue`

Skips the current iteration and continues with the next one.

### Example

```python
for number in range(1, 6):
    if number == 3:
        continue

    print(number)
```

---

## `pass`

Acts as a placeholder and performs no action.

### Example

```python
for number in range(5):
    pass
```

---

# Loop `else`

The `else` block executes only when the loop finishes normally.

If the loop ends because of a `break` statement, the `else` block is skipped.

### Example

```python
for number in range(1, 6):
    print(number)
else:
    print("Loop completed.")
```

---

# Comparison of Loop Control Statements

| Statement | Purpose |
|-----------|---------|
| `break` | Terminates the loop immediately |
| `continue` | Skips the current iteration |
| `pass` | Performs no action (placeholder) |

# Common Mistakes

- Forgetting that `range()` excludes the stop value.
- Forgetting to update the loop variable in a `while` loop.
- Confusing `break` with `continue`.
- Assuming the loop `else` block always executes.
- Creating infinite loops unintentionally.
- Incorrect indentation inside loop bodies.

---

# Key Takeaways

- Loops automate repetitive tasks.
- Use a `for` loop when the number of iterations is known.
