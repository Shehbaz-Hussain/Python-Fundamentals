# Frequently Asked Interview Questions

## 1. What is a loop?

A loop is a control structure that repeatedly executes a block of code until a specified condition is met or a sequence has been fully processed.

---

## 2. Why are loops used?

Loops help automate repetitive tasks, reduce duplicate code, improve readability, and make programs easier to maintain.

---

## 3. What are the two types of loops in Python?

- `for` loop
- `while` loop

---

## 4. When should you use a `for` loop?

Use a `for` loop when the number of iterations is known or when iterating over a sequence generated by `range()`.

Example:

```python
for number in range(5):
    print(number)
```

---

## 5. When should you use a `while` loop?

Use a `while` loop when repetition depends on a condition rather than a predetermined number of iterations.

Example:

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

## 6. What does the `range()` function do?

The `range()` function generates a sequence of integers that is commonly used with `for` loops.

Examples:

```python
range(5)
```

```python
range(1, 6)
```

```python
range(2, 11, 2)
```

---

## 7. Does `range()` include the stop value?

No.

The stop value is **excluded**.

Example:

```python
range(1, 5)
```

Produces:

```
1 2 3 4
```

---

## 8. What is a nested loop?

A nested loop is a loop placed inside another loop.

They are commonly used for:

- Pattern printing
- Multiplication tables
- Repeated row-and-column operations

---

## 9. What does the `break` statement do?

The `break` statement immediately terminates the nearest loop.

Example:

```python
for number in range(1, 11):
    if number == 6:
        break

    print(number)
```

---

## 10. What does the `continue` statement do?

The `continue` statement skips the remaining statements in the current iteration and moves directly to the next iteration.

---

## 11. What does the `pass` statement do?

The `pass` statement performs no action.

It is commonly used as a placeholder when a statement is syntactically required but no implementation is needed yet.

---

## 12. What is loop `else`?

The `else` block executes only if the loop finishes normally.

If the loop exits because of a `break` statement, the `else` block is skipped.

---

## 13. What is an infinite loop?

An infinite loop is a loop that never terminates because its condition always remains `True`.

Example:

```python
while True:
    print("Running")
```

---

## 14. How can you avoid infinite loops?

- Update loop variables correctly.
- Ensure the loop condition eventually becomes `False`.
- Use `break` when appropriate.

---

## 15. What are common uses of loops?

- Counting
- Repeating tasks
- Searching
- Menu-driven programs
- Password validation
- Pattern printing
- Number processing
- Simple games

# Common Interview Mistakes

- Forgetting that `range()` excludes the stop value.
- Forgetting to update the loop variable in a `while` loop.
- Confusing `break` and `continue`.
- Assuming the loop `else` block always executes.
- Writing unnecessarily complex nested loops.
- Using poor variable names such as `x` and `y` when descriptive names improve readability.

---

# Best Practices

- Use `for` loops when the number of iterations is known.
- Use `while` loops for condition-based repetition.
- Keep loop bodies concise and readable.
- Use meaningful variable names.
- Avoid unnecessary nesting.
- Use `break` and `continue` only when they improve clarity.
- Maintain consistent indentation.
- Test boundary cases such as zero, one, and empty iteration ranges.
