# Module 10 Assignment – Conditional Statements

## Overview

Welcome to the assignment for **Module 10 – Conditional Statements**.

This assignment gives you practical experience using Python decision-making statements to solve simple real-world problems. You will apply everything you have learned in this module by writing small Python programs that make decisions based on user input.

The exercises begin with simple `if` statements and gradually introduce more complex scenarios using:

- `if`
- `if...else`
- `if...elif...else`
- Nested `if` statements
- Comparison operators
- Logical operators

Complete every task by writing your own code without copying from examples.

# Programming Tasks

---

## Task 1 — Positive Number Checker

Ask the user to enter a number.

Display a message only if the number is positive.

---

## Task 2 — Even or Odd

Ask the user to enter an integer.

Display whether the number is even or odd.

---

## Task 3 — Voting Eligibility

Ask the user to enter their age.

Display whether the person is eligible to vote.

Assume the voting age is **18 years**.

---

## Task 4 — Pass or Fail

Ask the user to enter exam marks.

If the marks are **40 or above**, display:

```
Pass
```

Otherwise display:

```
Fail
```

---

## Task 5 — Larger Number

Ask the user to enter two numbers.

Display which number is larger.

If both numbers are equal, display an appropriate message.

---

## Task 6 — Grade Calculator

Ask the user to enter marks.

Display the grade according to the following rules.

| Marks | Grade |
|--------|-------|
| 90–100 | A |
| 80–89 | B |
| 70–79 | C |
| 60–69 | D |
| Below 60 | F |

---

## Task 7 — Temperature Advisor

Ask the user to enter the current temperature.

Display an appropriate message.

Example categories:

- Very Cold
- Cold
- Pleasant
- Hot

Choose suitable temperature ranges.

---

## Task 8 — Login Verification

Ask the user to enter:

- Username
- Password

Assume the correct values are:

Username:

```
admin
```

Password:

```
python123
```

Display:

- Login Successful

or

- Invalid Username or Password

---

## Task 9 — Largest of Three Numbers

Ask the user to enter three numbers.

Display the largest number.

---

## Task 10 — Simple Discount Calculator

Ask the user to enter:

- Product price

Apply the following discount rules.

| Price | Discount |
|--------|----------|
| 1000 or more | 20% |
| 500–999 | 10% |
| Below 500 | No discount |

Display:

- Original price
- Discount percentage
- Final price

# Common Mistakes to Avoid

Avoid these common errors.

- Forgetting the colon (`:`) after an `if` statement
- Incorrect indentation
- Using `=` instead of `==`
- Writing conditions in the wrong order
- Forgetting to test different inputs
- Using unnecessary nested conditions
- Choosing unclear variable names
- Printing incorrect messages
- Ignoring boundary values
- Not checking equal values where required
