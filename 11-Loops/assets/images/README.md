# Images

## Overview

This directory contains image assets used in **Module 10 – Conditional Statements**.

Images are optional but recommended because they make learning easier by helping learners visualize concepts and examples.

At the moment, this folder is intentionally kept as a placeholder. You can add images in the future as the repository grows.

---

# Recommended Images

You may add images such as:

- Flowchart of an `if` statement
- Flowchart of an `if...else` statement
- Flowchart of an `if...elif...else` statement
- Nested `if` decision flow
- Comparison operator diagram
- Logical operator truth table
- Decision-making process illustration
- Screenshots of example program outputs

---

# Suggested File Names

Use clear and descriptive names for image files.

Examples:

```
if-statement-flowchart.png
if-else-flowchart.png
if-elif-else-flowchart.png
nested-if-flowchart.png
comparison-operators.png
logical-operators-truth-table.png
decision-making-diagram.png
example-output-01.png
example-output-02.png
```

---

# Image Guidelines

When adding images to this folder:

- Use high-quality images.
- Keep diagrams simple and easy to understand.
- Use readable fonts.
- Avoid unnecessary decorations.
- Use consistent colors and formatting across all diagrams.
- Ensure screenshots are clear and properly cropped.

---

# Supported Image Formats

Recommended formats include:

- PNG
- JPG
- JPEG
- SVG (for diagrams)

PNG is recommended for diagrams because it provides sharp text and clear graphics.

- Keep image file names descriptive.
- Avoid spaces in file names.
- Prefer lowercase letters with hyphens.
- Optimize images to reduce file size while maintaining readability.
