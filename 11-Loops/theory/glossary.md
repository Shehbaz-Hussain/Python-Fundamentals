# Glossary — Loops

This glossary defines the important terms introduced in the **Loops** module. Understanding these terms will help you read Python documentation, write correct programs, and communicate using standard programming terminology.

---

# A

## Algorithm

A sequence of well-defined steps used to solve a problem or perform a task.

Example:

```text
Start
↓
Read input
↓
Repeat calculations
↓
Display result
↓
End
```

---

# B

## `break`

A loop control statement that immediately terminates the nearest enclosing loop.

Example:

```python
for number in range(5):
    if number == 3:
        break
    print(number)
```

Output:

```text
0
1
2
```

---

# C

## Condition

A Boolean expression that evaluates to either `True` or `False`.

Example:

```python
count <= 5
```

---

## Counter Variable

A variable whose value changes during loop execution to keep track of progress.

Example:

```python
count = 1
```

---

## `continue`

A loop control statement that skips the remaining statements in the current iteration and proceeds to the next iteration.

Example:

```python
for number in range(5):
    if number == 2:
        continue
    print(number)
```

Output:

```text
0
1
3
4
```

---

# E

## `else` (Loop)

An optional block associated with a `for` or `while` loop that executes only if the loop finishes normally without a `break`.

Example:

```python
for number in range(3):
    print(number)
else:
    print("Completed")
```

---

# F

## `for` Loop

A loop that iterates over a sequence of values.

Example:

```python
for number in range(5):
    print(number)
```

---

# I

## Infinite Loop

A loop that never terminates because its condition never becomes `False` or there is no terminating condition.

Example:

```python
while True:
    print("Running...")
```

---

## Iteration

One complete execution of a loop body.

Example:

```python
for number in range(3):
    print(number)
```

This loop performs three iterations.

---

# L

## Loop

A control flow structure that repeatedly executes a block of code.

Python provides two primary loop statements:

- `while`
- `for`

---

## Loop Body

The indented block of code executed during each iteration.

Example:

```python
for number in range(3):
    print(number)
```

The statement:

```python
print(number)
```

is the loop body.

---

## Loop Condition

A Boolean expression that determines whether a `while` loop should continue executing.

Example:

```python
while count <= 5:
```

The condition is:

```python
count <= 5
```

---

## Loop Control Statement

A statement that changes the normal flow of loop execution.

Examples:

- `break`
- `continue`
- `pass`

---

# N

## Nested Loop

A loop placed inside another loop.

Example:

```python
for row in range(2):
    for column in range(3):
        print(row, column)
```

---

# P

## `pass`

A null statement that performs no action.

It is commonly used as a placeholder for future code.

Example:

```python
for number in range(5):
    pass
```

---

# R

## `range()`

A built-in Python function that generates a sequence of integers.

Example:

```python
range(1, 6)
```

Generates:

```text
1 2 3 4 5
```

The `stop` value is excluded.

---

# S

## Sequence

An ordered collection of values that can be iterated over.

In this module, the sequence generated by `range()` is the primary example.

---

## Step

The amount by which values increase or decrease in `range()`.

Example:

```python
range(2, 11, 2)
```

Produces:

```text
2 4 6 8 10
```

The step is `2`.

---

## Stop Value

The ending value supplied to `range()`.

The stop value is **not included** in the generated sequence.

Example:

```python
range(1, 6)
```

Produces:

```text
1 2 3 4 5
```

---

# W

## `while` Loop

A loop that repeats while its condition remains `True`.

Example:

```python
count = 1

while count <= 3:
    print(count)
    count = count + 1
```
