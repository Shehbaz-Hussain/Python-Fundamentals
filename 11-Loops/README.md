# Module 11 — Loops

## Module Overview

Loops are one of the most powerful control flow mechanisms in programming. They allow a program to execute the same block of code repeatedly without writing duplicate code.

In previous modules, you learned how to make decisions using conditional statements. In this module, you will learn how to repeat those decisions and actions efficiently using loops.

Python provides two primary looping constructs:

- `while` loop
- `for` loop

You will also learn how to control loop execution using:

- `break`
- `continue`
- `pass`
- `else` with loops

By the end of this module, you will be able to write programs that automate repetitive tasks, process sequences of values, and solve more complex programming problems.

After completing this module, you will be able to:
