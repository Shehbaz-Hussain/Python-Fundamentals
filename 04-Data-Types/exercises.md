# Exercises: Data Types

## Easy Exercises

1. Create a variable `age` and assign your age to it.
2. Create a variable `name` and assign your name to it.
3. Create a list of three fruits.
4. Create a tuple with three coordinates.
5. Create a dictionary with `name` and `age` keys.
6. Print the type of `True`.
7. Convert the string `"10"` to an integer.
8. Convert the integer `7` to a float.
9. Create a set from a list with duplicates.
10. Create a variable `is_student` and assign `True`.
11. Store a sentence in a string variable.
12. Create a range from `0` to `5`.
13. Create a list and add one element to it.
14. Try to change the first character of a string. Explain what happens.
15. Create a dictionary of three books and their authors.
16. Store a price as a float.
17. Create a tuple containing your favorite colors.
18. Create a `bytes` object from a string.
19. Create a `None` value and print it.
20. Create a list of booleans.

## Medium Exercises

2. Create a list of numbers and find their average.
3. Convert a list to a tuple and print the result.
4. Use a set to remove duplicates from a list.
5. Write code to compare two values using `==` and `is`.
6. Create a nested dictionary for a school timetable.
7. Use `range` to print numbers from 1 to 10.
8. Create a function that accepts a list and returns its length.
9. Create a program that stores whether a user is online as a boolean.
10. Create a dictionary of product names and prices.
11. Write code that uses `bytes` and prints its length.
12. Create a tuple and try to modify it. Explain the result.
13. Merge two dictionaries into one.
14. Write code to check whether an item exists in a set.
15. Create a list of strings and join them into one sentence.
16. Store a user profile as a dictionary.
17. Create a list of student scores and find the highest score.
18. Use `frozenset` to store a fixed set of values.
19. Convert a float to an integer and explain the result.
20. Create a program that uses `bool` values in a condition.

15. Build a simple recommendation system using lists and dictionaries.
