# Quiz: Data Types

## Instructions

Choose the best answer for each question.

## Questions

### Easy

1. Which of the following is a Python integer?
   - A. `3.14`
   - B. `"3"`
   - C. `3`
   - D. `True`
   - Answer: C

2. Which type is used for text?
   - A. `list`
   - B. `str`
   - C. `dict`
   - D. `set`
   - Answer: B

3. Which of the following is mutable?
   - A. `tuple`
   - B. `str`
   - C. `list`
   - D. `int`
   - Answer: C

4. What does `None` represent?
   - A. zero
   - B. empty string
   - C. missing value
   - D. false
   - Answer: C

5. Which of these is a dictionary?
   - A. `[1, 2, 3]`
   - B. `(1, 2, 3)`
   - C. `{1: 'a', 2: 'b'}`
   - D. `{'a', 'b'}`
   - Answer: C

6. What is the result of `type(True)`?
   - A. `<class 'int'>`
   - B. `<class 'bool'>`
   - C. `<class 'str'>`
   - D. `<class 'list'>`
   - Answer: B

7. Which type stores unique values only?
   - A. `list`
   - B. `tuple`
   - C. `set`
   - D. `dict`
   - Answer: C

8. Which type is ordered and immutable?
   - A. `list`
   - B. `tuple`
   - C. `dict`
   - D. `set`
   - Answer: B

9. What does `==` compare?
   - A. identity
   - B. memory address
   - C. values
   - D. type only
   - Answer: C

10. What does `is` compare?
   - A. values
   - B. identity
   - C. length
   - D. hash
   - Answer: B

### Medium

11. Which statement is true about Python's typing system?
   - A. It is statically typed
   - B. It is weakly typed
   - C. It is dynamically typed
   - D. It is not typed
   - Answer: C

12. Which of the following is an immutable collection?
   - A. `list`
   - B. `dict`
   - C. `tuple`
   - D. `set`
   - Answer: C

13. What is the output of `len('Python')`?
   - A. 5
   - B. 6
   - C. 7
   - D. 8
   - Answer: B

14. Which data type is best for storing key-value pairs?
   - A. `list`
   - B. `tuple`
   - C. `dict`
   - D. `set`
   - Answer: C

15. Which of these is a float?
   - A. `2`
   - B. `2.0`
   - C. `True`
   - D. `"2.0"`
   - Answer: B

16. Which type is used for binary data?
   - A. `str`
   - B. `bytes`
   - C. `bool`
   - D. `dict`
   - Answer: B

17. What is the main difference between `list` and `tuple`?
   - A. one stores text only
   - B. one is mutable, the other is not
   - C. one cannot be printed
   - D. one is numeric only
   - Answer: B

18. Which of the following creates an empty dictionary?
   - A. `[]`
   - B. `()`
   - C. `{}`
   - D. `set()`
   - Answer: C

19. What does `frozenset` provide?
   - A. a mutable set
   - B. a set with duplicate values
   - C. an immutable set
   - D. a mapping
   - Answer: C

20. Why are strings immutable?
   - A. to allow fast changes
   - B. to make them safer and easier to optimize
   - C. because they are sets
   - D. because they are dictionaries
   - Answer: B

### Hard

21. Which is the best choice for a fixed record like a point `(x, y)`?
   - A. `list`
   - B. `dict`
   - C. `tuple`
   - D. `set`
   - Answer: C

22. Which collection is best when order does not matter and duplicates should be removed?
   - A. `list`
   - B. `tuple`
   - C. `set`
   - D. `dict`
   - Answer: C

23. Which statement about `range` is correct?
   - A. it stores all values in memory at once
   - B. it generates values lazily
   - C. it stores key-value pairs
   - D. it is mutable
   - Answer: B

24. Which of the following is a valid complex number?
   - A. `3 + 2j`
   - B. `3/2`
   - C. `True`
   - D. `"3+2j"`
   - Answer: A

25. What is the best reason to use `bytes` instead of `str` for binary data?
   - A. `bytes` can contain only letters
   - B. `bytes` is mutable
   - C. `bytes` stores raw binary values
   - D. `bytes` is faster for text
   - Answer: C

26. Which type is commonly used for feature vectors in AI systems?
   - A. `set`
   - B. `dict`
   - C. `list`
   - D. `None`
   - Answer: C

27. Why does `bool` inherit from `int` in Python?
   - A. because it is a subclass of `int`
   - B. because `True` and `False` are values of `int`
   - C. because booleans are mutable
   - D. because Python treats them as strings
   - Answer: B

28. Which type should be used to represent a fixed set of constant values?
   - A. `set`
   - B. `frozenset`
   - C. `list`
   - D. `dict`
   - Answer: B

29. Which structure is best for filtering duplicate values from a list?
   - A. `set`
   - B. `tuple`
   - C. `dict`
   - D. `range`
   - Answer: A

30. What is the main reason to use `memoryview`?
   - A. to handle text
   - B. to view binary data efficiently
   - C. to store key-value pairs
   - D. to create immutable lists
   - Answer: B
