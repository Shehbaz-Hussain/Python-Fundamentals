# Assignments: Data Types

## Assignment 1: Student Profile Builder

### Problem Statement
Create a program that stores the details of a student using a dictionary.

### Objectives
- use a dictionary for structured data
- store name, age, courses, and GPA

### Requirements
- use strings, integers, floats, and booleans
- print the profile neatly

### Expected Output
Example output:

```text
Name: Amina
Age: 20
Courses: ['Math', 'Science']
GPA: 3.7
Active: True
```

### Hints
Use a dictionary and print each key-value pair.

## Assignment 2: Shopping List Manager

### Problem Statement
Build a simple shopping list app using a list.

### Objectives
- append and remove items
- display the list

### Requirements
- use a list of strings
- allow at least three items

### Expected Output
```text
['milk', 'bread', 'eggs']
```

### Hints
Use `.append()` and `.remove()`.

## Assignment 3: Data Type Analyzer

### Problem Statement
Create a function that takes a value and prints its type and whether it is mutable.

### Objectives
- practice `type()` and mutability concepts

### Requirements
- support at least 5 sample values

### Expected Output
```text
Value: 10
Type: <class 'int'>
Mutable: False
```

### Hints
Use `type()` and a small conditional structure.
