# Comparison Operators

Comparison operators are used to compare two values or expressions.

The result of every comparison is a Boolean value.

- `True` means the comparison is true.
- `False` means the comparison is false.

## Comparison Operators

| Operator | Description |
|----------|-------------|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or equal to |
| `<=` | Less than or equal to |

## Syntax

```python
value1 operator value2
```

## Examples

```python
10 == 10
10 != 5
20 > 15
8 < 12
18 >= 18
7 <= 9
```

## Boolean Results

Comparison operators always return one of the following values:

- `True`
- `False`

## Summary

Comparison operators compare two values and return a Boolean result. They are an essential part of Python programming and are widely used to evaluate conditions.
