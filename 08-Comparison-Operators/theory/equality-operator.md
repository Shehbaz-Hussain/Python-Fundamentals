# Equality Operator (`==`)

The equality operator (`==`) is used to check whether two values are equal.

If both values are equal, the result is `True`.

If the values are not equal, the result is `False`.

## Syntax

```python
value1 == value2
```

## Examples

```python
10 == 10
```

Result:

```python
True
```

```python
10 == 20
```

Result:

```python
False
```

```python
5 == 5
```

Result:

```python
True
```

## Key Points

- Uses the `==` operator.
- Compares two values or expressions.
- Returns a Boolean value.
- Does not change the values being compared.

## Summary

The equality operator checks whether two values are equal and returns either `True` or `False`.
