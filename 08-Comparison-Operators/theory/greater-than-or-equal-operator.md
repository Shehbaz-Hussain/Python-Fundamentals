# Greater Than or Equal To Operator (`>=`)

The greater than or equal to operator (`>=`) is used to check whether the value on the left is greater than or equal to the value on the right.

If the left value is greater than the right value, the result is `True`.

If both values are equal, the result is also `True`.

Otherwise, the result is `False`.

## Syntax

```python
value1 >= value2
```

## Examples

```python
20 >= 10
```

Result:

```python
True
```

```python
15 >= 15
```

Result:

```python
True
```

```python
10 >= 20
```

Result:

```python
False
```

## Key Points

- Uses the `>=` operator.
- Compares two values or expressions.
- Returns a Boolean value.
- Greater or equal values produce `True`.

## Summary

The greater than or equal to operator checks whether one value is greater than or equal to another and returns either `True` or `False`.
