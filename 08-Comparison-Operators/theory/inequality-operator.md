# Inequality Operator (`!=`)

The inequality operator (`!=`) is used to check whether two values are not equal.

If the values are different, the result is `True`.

If both values are equal, the result is `False`.

## Syntax

```python
value1 != value2
```

## Examples

```python
10 != 20
```

Result:

```python
True
```

```python
10 != 10
```

Result:

```python
False
```

```python
5 != 8
```

Result:

```python
True
```

## Key Points

- Uses the `!=` operator.
- Compares two values or expressions.
- Returns a Boolean value.
- Does not modify the values being compared.

## Summary

The inequality operator checks whether two values are different and returns either `True` or `False`.
