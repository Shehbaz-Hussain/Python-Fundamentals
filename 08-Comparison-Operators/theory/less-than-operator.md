# Less Than Operator (`<`)

The less than operator (`<`) is used to check whether the value on the left is less than the value on the right.

If the left value is less than the right value, the result is `True`.

Otherwise, the result is `False`.

## Syntax

```python
value1 < value2
```

## Examples

```python
10 < 20
```

Result:

```python
True
```

```python
20 < 10
```

Result:

```python
False
```

```python
15 < 15
```

Result:

```python
False
```

## Key Points

- Uses the `<` operator.
- Compares two values or expressions.
- Returns a Boolean value.
- Equal values produce `False`.

## Summary

The less than operator checks whether one value is less than another and returns either `True` or `False`.
