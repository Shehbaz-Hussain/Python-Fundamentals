# Comparison Operators Syntax

Comparison operators are written between two values or expressions.

## General Syntax

```python
value1 operator value2
```

## Equality

```python
value1 == value2
```

## Inequality

```python
value1 != value2
```

## Greater Than

```python
value1 > value2
```

## Less Than

```python
value1 < value2
```

## Greater Than or Equal To

```python
value1 >= value2
```

## Less Than or Equal To

```python
value1 <= value2
```

## Using Variables

```python
number1 > number2
```

```python
age >= 18
```

```python
marks == 90
```

```python
salary != 50000
```

## Boolean Result

Every comparison expression returns either:

```python
True
```

or

```python
False
```
