# Comparison Operators Examples

## Equality Operator

```python
10 == 10
```

Output

```python
True
```

---

## Inequality Operator

```python
10 != 20
```

Output

```python
True
```

---

## Greater Than Operator

```python
20 > 10
```

Output

```python
True
```

---

## Less Than Operator

```python
10 < 20
```

Output

```python
True
```

---

## Greater Than or Equal To Operator

```python
15 >= 15
```

Output

```python
True
```

---

## Less Than or Equal To Operator

```python
15 <= 20
```

Output

```python
True
```

---

## Comparing Variables

```python
age = 18
limit = 16

age > limit
```

Output

```python
True
```

---

## Comparing User Input

```python
number = int(input("Enter a number: "))

number > 100
```

Possible Output

```python
True
```

or

```python
False
```

---
