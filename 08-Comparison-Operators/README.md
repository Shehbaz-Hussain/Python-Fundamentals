# 08 - Comparison Operators

## Overview

This module introduces Python comparison operators. Comparison operators are used to compare two values or expressions. The result of every comparison is a Boolean value: `True` or `False`.

Comparison operators are commonly used when making decisions in programs and form the foundation for conditional statements that will be covered in later modules.

---

## Learning Objectives

After completing this module, you will be able to:

- Understand comparison operators in Python.
- Compare numbers, variables, and expressions.
- Use equality and inequality operators.
- Use greater than and less than operators.
- Use greater than or equal to and less than or equal to operators.
- Interpret Boolean results.
- Write simple comparison-based programs.

## Comparison Operators

| Operator | Description |
|----------|-------------|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or equal to |
| `<=` | Less than or equal to |
