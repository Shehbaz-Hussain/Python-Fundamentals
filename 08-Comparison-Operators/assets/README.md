# Assets

This directory contains images, diagrams, screenshots, and other resources related to the **Comparison Operators** module.

## Suggested Files

- `comparison-operators.png`
- `comparison-table.png`
- `comparison-flowchart.png`
- `examples-output.png`
- `quiz-diagram.png`

## Purpose

The files in this directory can be used to:

- Illustrate comparison operators visually.
- Show example program outputs.
- Explain Boolean results.
- Support documentation and learning materials.

> This folder is optional. The module works correctly even if no asset files are added.
