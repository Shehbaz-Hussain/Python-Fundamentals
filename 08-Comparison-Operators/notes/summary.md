# Module Summary

In this module, you learned how to compare values using Python comparison operators.

## Topics Covered

- Comparison operators
- Equality operator (`==`)
- Inequality operator (`!=`)
- Greater than operator (`>`)
- Less than operator (`<`)
- Greater than or equal to operator (`>=`)
- Less than or equal to operator (`<=`)
- Boolean values (`True` and `False`)
- Comparing variables
- Comparing user input
- Writing simple comparison programs

---

## Comparison Operators

| Operator | Description |
|----------|-------------|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or equal to |
| `<=` | Less than or equal to |

---

## Key Points

- Comparison operators compare two values or expressions.
- Every comparison returns a Boolean value.
- A Boolean result is either `True` or `False`.
- Comparison operators do not modify values.
- Convert user input to the correct data type before comparing it.
- Use meaningful variable names to improve code readability.
