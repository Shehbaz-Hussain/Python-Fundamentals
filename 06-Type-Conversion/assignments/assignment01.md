# Objective

The objective of this assignment is to help students understand and apply
basic type conversion techniques in Python. Students will practice using
built-in conversion functions to convert values between different data
types and observe the results.

---

# Background

In Python, the `input()` function always returns data as a string. Before
performing mathematical operations or other data processing tasks, it is
often necessary to convert the input into the appropriate data type.

Python provides several built-in functions for explicit type conversion:

- `int()`
- `float()`
- `str()`
- `bool()`
- `type()`

Understanding these functions is essential for writing correct and reliable
Python programs.
