- **Assignment:** 02

# Objective

The objective of this assignment is to apply type conversion concepts in
real-world programming scenarios. Students will practice converting user
input into suitable data types and using converted values in calculations.

---

# Background

Real-world applications receive data in different formats. For example:

- User input is received as text.
- Prices require decimal values.
- Ages and quantities require whole numbers.
- Display messages often require string conversion.

Correct type conversion allows programs to process data accurately.

---

# Learning Outcomes

After completing this assignment, students will be able to:

- Convert user input into integers and floating-point numbers.
- Perform calculations using converted data.
- Understand the role of different conversion functions.
- Apply type conversion in practical programs.

---

# Tasks

## Task 1: Student Marks Calculator (8 Marks)

Create a program that:

1. Takes marks for three subjects from the user.
2. Converts all inputs into integers.
3. Calculates:
   - Total marks
   - Average marks
4. Displays:
   - Individual marks
   - Total
   - Average
   - Data types of calculated values

---

## Task 2: Shopping Cost Calculator (8 Marks)

Create a program that:

1. Takes the price of three products from the user.
2. Converts prices into floating-point numbers.
3. Calculates the total shopping cost.
4. Displays:
   - Product prices
   - Total bill
   - Data types of values

---

## Task 3: Temperature Converter (5 Marks)

Create a program that:

1. Takes temperature in Celsius from the user.
2. Converts the input into a float.
3. Converts Celsius into Fahrenheit using:

```
Fahrenheit = (Celsius × 9 / 5) + 32
```

4. Displays both temperatures.

---

## Task 4: Data Type Demonstration (4 Marks)

Create a program that:

1. Stores different values:

- Integer
- Float
- String
- Boolean

2. Displays:

- Value
- Data type
