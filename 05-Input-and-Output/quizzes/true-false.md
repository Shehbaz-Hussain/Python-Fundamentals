# True or False Questions

## Question 1

`print()` is used to read input from the user.

### Answer

False

### Explanation

`print()` is used for output, not input.

---

## Question 2

`input()` always returns a string.

### Answer

True

### Explanation

This is an important behavior of the `input()` function.

---

## Question 3

The `sep` parameter changes the end of a printed line.

### Answer

False

### Explanation

`sep` changes the separator between values, while `end` changes the line ending.

---

## Question 4

`f-strings` can insert variables into a string.

### Answer

True

### Explanation

`f-strings` make string formatting simple and readable.

---

## Question 5

Type conversion is not necessary when using `input()`.

### Answer

False

### Explanation

Type conversion is often necessary for numbers and other data types.
