# Output Prediction Questions

## Question 1

What is the output of the following code?

```python
print("Hello")
print("World")
```

### Answer

```text
Hello
World
```

---

## Question 2

What is the output of the following code?

```python
print("A", "B", "C", sep="-")
```

### Answer

```text
A-B-C
```

## Question 3

What is the output of the following code?

```python
print("Hello", end="!")
print("Python")
```

### Answer

```text
Hello!Python
```

## Question 4

What is the output of the following code?

```python

### Answer

```
