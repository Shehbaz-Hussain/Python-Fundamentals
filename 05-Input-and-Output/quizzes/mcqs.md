# MCQs: Input and Output in Python

## Question 1

Which function is used to display output in Python?

A. `input()`  
B. `print()`  
C. `format()`  
D. `type()`

### Answer

B. `print()`

### Explanation

The `print()` function is used to display output on the screen.

---

## Question 2

Which function is used to receive input from the user?

A. `display()`  
B. `read()`  
C. `input()`  
D. `show()`

### Answer

C. `input()`

### Explanation

The `input()` function reads a value entered by the user.

---

## Question 3

What does `input()` return by default?

A. Integer  
B. Float  
C. String  
D. Boolean

### Answer

C. String

### Explanation

`input()` always returns a string value.

---

## Question 4

Which parameter controls the separator between values in `print()`?

A. `end`  
B. `sep`  
C. `split`  
D. `join`

### Answer

B. `sep`

### Explanation

`sep` changes the separator used between printed values.

---

## Question 5

Which formatting method is considered modern and recommended?

A. `%` formatting  
B. `format()`  
C. `f-strings`  
D. `join()`

### Answer

C. `f-strings`

### Explanation

`f-strings` are concise and easy to read.
