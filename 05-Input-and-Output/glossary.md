# Glossary: Input and Output in Python

## Terms and Definitions

### Input
Input is data received by a program from a user or another source.

### Output
Output is information displayed by a program to the user.

### `print()`
The built-in function used to display output on the screen.

### `input()`
The built-in function used to receive input from the user.

### Prompt
A message shown to the user before asking for input.

### String
A sequence of characters enclosed in quotes.

### Integer
A whole number such as `10` or `-3`.

### Float
A decimal number such as `3.14`.

### Type Conversion
Changing one data type into another, such as converting a string into an integer.

### `sep`
A parameter of `print()` that controls the separator between values.

### `end`
A parameter of `print()` that controls what is printed at the end of the output line.

### Escape Character
A special character used inside strings to represent actions such as a new line or tab.

### f-string
A modern string formatting technique that allows variables to be inserted into strings using braces.

### Console Application
A program that runs in the terminal or command line.

### Interactive Program
A program that responds to user input while running.

---

## Quick Memory Tip

Think of input as “getting information” and output as “showing information.”
