# Print Function Flowchart

```text
+----------------------------+
| Start                      |
+----------------------------+
| Call print()               |
+----------------------------+
| Prepare values to display |
+----------------------------+
| Apply sep and end rules   |
+----------------------------+
| Display output on screen  |
+----------------------------+
| End                        |
+----------------------------+
```

## Key Idea

`print()` sends information to the console so users can see it.
