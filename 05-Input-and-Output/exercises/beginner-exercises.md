# Beginner Exercises: Input and Output

These exercises will help you practice the basic concepts of input and output in Python.

## Exercise 1

Write a program that asks the user for their name and prints a greeting.

### Example

```python
name = input("Enter your name: ")
print(f"Hello, {name}!")
```
