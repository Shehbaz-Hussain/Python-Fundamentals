# Solutions: Input and Output Exercises

## Beginner Exercise 1

```python
name = input("Enter your name: ")
print(f"Hello, {name}!")
```

## Beginner Exercise 2

```python
age = int(input("Enter your age: "))
print(f"Next year you will be {age + 1} years old")
```

## Beginner Exercise 3

```python
first_number = int(input("Enter first number: "))
second_number = int(input("Enter second number: "))
print(first_number + second_number)
```

## Beginner Exercise 4

```python
favorite_color = input("Enter your favorite color: ")
print(f"Your favorite color is {favorite_color}")
```

## Beginner Exercise 5

```python
print("A", "B", "C", sep="-")
```

## Intermediate Exercise 1

```python
name = input("Enter student name: ")
age = input("Enter student age: ")
grade = input("Enter student grade: ")
print(f"Name: {name}")
print(f"Age: {age}")
print(f"Grade: {grade}")
```

## Intermediate Exercise 2

```python
height_meters = float(input("Enter height in meters: "))
height_centimeters = height_meters * 100
print(f"Height in centimeters: {height_centimeters}")
```

## Intermediate Exercise 3

```python
import math
radius = float(input("Enter radius: "))
area = math.pi * radius ** 2
print(f"Area: {area}")
```

## Intermediate Exercise 4

```python
first_name = input("Enter first name: ")

```
