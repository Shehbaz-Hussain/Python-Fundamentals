# Intermediate Exercises: Input and Output

These problems are slightly more challenging and require you to combine multiple concepts.

## Exercise 1

Write a program that asks for a student’s name, age, and grade. Print the information in a neat format.

## Exercise 2

Create a program that asks the user for their height in meters and converts it into centimeters.

## Exercise 3

Write a program that asks the user for the radius of a circle and prints the area using the formula $A = \pi r^2$.

## Exercise 4

Create a program that asks the user for their first name and last name, then prints the initials.

## Exercise 5

Write a program that asks the user to enter three words and prints them in a single line separated by commas.
