# Challenge Problems: Input and Output

These problems are designed for learners who want to apply input and output in more realistic scenarios.

## Challenge 1

Build a mini calculator that supports addition, subtraction, multiplication, and division.

## Challenge 2

Create a program that asks the user for a temperature in Celsius and converts it to Fahrenheit.

## Challenge 3

Write a program that asks for a username and password, then prints a welcome message if both are valid.

## Challenge 4

Create a simple quiz program that asks three questions and shows the score at the end.

## Challenge 5

Build a console-based menu system that lets the user choose from different operations and prints the result.
